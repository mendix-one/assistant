# AI Approaches for Resource Planning

## Context

Research on AI/ML techniques that complement multi-dimensional analysis for solving resource planning problems in Samsung Electronics Device Solutions (DS) Division. This extends the MDA-based resource planning research with AI-driven optimization, prediction, and autonomous decision-making.

## Relationship to MDA Research

```
Multi-Dimensional Analysis (MDA)          AI/ML Approaches
+-----------------------------+          +-----------------------------+
| Understand & Visualize      |          | Predict & Optimize          |
| - OLAP cube operations      |  <---->  | - Forecasting models        |
| - Snowflake schema          |          | - Scheduling solvers        |
| - Kibana dashboards         |          | - Autonomous agents         |
| - What-if scenario compare  |          | - What-if scenario generate |
+-----------------------------+          +-----------------------------+
         MDA answers "what is"           AI answers "what should be"
```
